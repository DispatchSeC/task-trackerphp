Создал на PHP 8.x и CodeIgniter 4.

git clone https://github.com/DispatchSeC/task-trackerphp.git

cd task-trackerphp

composer install

cp .env.example .env

php spark serve

Настройка БД:
Откройте созданный файл .env.example

Укажите параметры подключения к БД в разделе database.default.

Приложение доступно по адресу: http://localhost:8080.
