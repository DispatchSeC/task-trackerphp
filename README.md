Создал на PHP 8.x и CodeIgniter 4.

git clone https://github.com/DispatchSeC/task-trackerphp.git

cd task-trackerphp

composer install

cp .env.example .env

php spark serve

Настройка БД:
Откройте файл .env.example

Укажите параметры подключения к БД в  database.default.

Адрес: http://localhost:8080.
